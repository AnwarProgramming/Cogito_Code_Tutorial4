# Vue.js (This is Vue3)
Vue allows us to create single-File Component(SFC). Where we need to fetch only the changed portion instead of whole application again.

### We are creating Frontend Server using Vue3 CLI

> npm create vue@latest
It is the command used to scaffold a new Vue.js project using the official Vue project generator. we also need Pinia and Router

> cd frontend 
frontend is the project name you put while creating vue@latest

> npm install
It is used to install node.js module which are there

> npm run dev
It is used to run the application
